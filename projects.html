PuyoJS

The classic game of PuyoPuyo recreated in JavaScript! This game is one of my first tests with P5JS and has a few rough patches. The collision and nearby Puyo detection were an interesting challenges because they required me to think of it from a grid-based mindset instead of the more open methods common in game engines, such as Unity3D.

  • Spring/Summer 2019
  • Main Technologies: P5JS, JS, Adobe Illustrator
  • Repo Link

Form2Role Bot

A Discord bot created to assign roles to students in a school Discord Server. The idea was that students would fill out a Google Form, then the bot would automatically assign them those roles so that each student had access to text channels for those classes. This would make it easy for them to ask questions and connect with others in the class.

  • Main Technologies: Discord.NET, C#, Google Sheets API
  • Originally Created for the Congressional App Challenge in Fall 2018
  • Repo Link
